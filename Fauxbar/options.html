Quick Delete
If Quick Delete is enabled and you press Delete:
 • any highlighted history results will be deleted from Chrome's history without prompt.
 • any highlighted search queries will be deleted from Fauxbar's query history.
Highlighted bookmarks remain safe and will not be deleted, though they may temporarily disappear for a moment.

Database
Options
To backup your Fauxbar options:
1. Copy all the text in the box below to your clipboard.
2. Paste the text into a new plain-text document somewhere, then save it for later.
Backups contain all cosmetic and functionality settings you've chosen, including your site tiles, Fauxbar search engines, URL keywords, and blacklist.
However, backups do not contain your history items, bookmarks, or search queries.
To restore your Fauxbar options from an existing backup:
1. Open the document containing the backup text.
2. Select all the text, then copy it to your clipboard.
3. Paste the text into the box below.

Memory Helper
To minimize Fauxbar's memory usage, Fauxbar Memory Helper is a Chrome extension that quietly reloads Fauxbar when your computer has been idle for ~60 seconds, as long as the main Fauxbar or Fauxbar Options pages aren't open.
